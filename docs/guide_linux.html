Guía de Instalación para Linux

Instala Atlas Interactivo en distribuciones Linux usando AppImage. No requiere permisos root ni instalación del sistema.

Distribuciones Compatibles

Atlas Interactivo funciona en cualquier distribución Linux moderna de 64-bit gracias a AppImage.

Ubuntu

20.04 LTS o superior

Debian

11 (Bullseye) o superior

Fedora

34 o superior

Arch Linux

Cualquier versión reciente

# Verifica tu distribución y versión cat /etc/os-release # Verifica si es 64-bit uname -m # Debe mostrar: x86_64 o aarch64

Requisitos del Sistema

Arquitectura

x86_64 (64-bit)

Memoria RAM

Mínimo 2 GB

Espacio en Disco

15 GB libres

Entorno Gráfico

GNOME, KDE, XFCE, etc.

Nota sobre AppImage: No requiere dependencias especiales ni instalación en el sistema. Todo está contenido en un solo archivo ejecutable.

Métodos de Instalación

Elige el método que prefieras:

GUI (Recomendado)
Terminal
Usando wget

Método Gráfico (Recomendado)

1

Descarga desde el navegador

Visita nuestra página de descargas y haz clic en "Descargar para Linux".

2

Hazlo ejecutable

Navega a la carpeta de descargas (normalmente ~/Descargas), haz clic derecho en AtlasInstaller.AppImage y selecciona:

Propiedades → Permisos → ☑ Permitir ejecutar como programa

O usa el terminal:

$ cd ~/Descargas
$ chmod +x AtlasInstaller.AppImage
3

Ejecuta el instalador

Doble clic en AtlasInstaller.AppImage o usa el terminal:

$ ./AtlasInstaller.AppImage

Método por Terminal

1. Abre una terminal

Presiona Ctrl + Alt + T o busca "Terminal" en tu menú de aplicaciones.

2. Navega a la carpeta de descargas

$ cd ~/Descargas

3. Descarga el instalador

$ wget "https://drive.google.com/uc?id=TU_ID_LINUX&export=download" -O AtlasInstaller.AppImage
# Reemplaza TU_ID_LINUX con el ID real de Drive

4. Hazlo ejecutable

$ chmod +x AtlasInstaller.AppImage

5. Ejecuta el instalador

$ ./AtlasInstaller.AppImage

Descarga Directa con wget

Si prefieres un comando único:

# Descarga y ejecuta en un solo comando:
$ cd ~ && wget -O AtlasInstaller.AppImage "URL_DE_DESCARGA" && chmod +x AtlasInstaller.AppImage && ./AtlasInstaller.AppImage
Seguridad: Solo descarga de fuentes confiables. Verifica que la URL sea de nuestro sitio oficial.
1

Descarga el AppImage

El archivo AtlasInstaller.AppImage es un ejecutable portable de ~5 MB que contiene todo lo necesario.

Nombre: AtlasInstaller.AppImage
Tamaño: ~5 MB
Tipo: AppImage (executable Linux)
SHA256: [checksum disponible en la página]
Verificación de integridad: Después de descargar, puedes verificar el checksum:
$ sha256sum AtlasInstaller.AppImage
# Compara el resultado con el publicado en nuestra página
2

Configura Permisos de Ejecución

Linux requiere permisos explícitos para ejecutar archivos. Si usas GUI, haz clic derecho → Propiedades → Permisos.

En terminal (recomendado):

$ chmod +x AtlasInstaller.AppImage
# Esto agrega el permiso de ejecución

$ ls -la AtlasInstaller.AppImage
# Deberías ver: -rwxr-xr-x ... AtlasInstaller.AppImage
# La 'x' indica que es ejecutable
Error común: Si ves "Permiso denegado" al ejecutar, significa que no configuraste los permisos correctamente. Usa chmod +x.
3

Ejecuta el Instalador

$ ./AtlasInstaller.AppImage
# El './' es necesario para ejecutar desde el directorio actual

Interfaz del instalador: Verás una ventana con:

  • Selector de carpeta de instalación
  • Barra de progreso de descarga (13 GB total)
  • Información de espacio disponible
  • Opciones de instalación
Ubicación recomendada:
~/AtlasInteractivo/ (en tu carpeta home)
~/Apps/AtlasInteractivo/ (si tienes carpeta Apps)
/opt/AtlasInteractivo/ (si quieres acceso global, necesita sudo)
4

Descarga Automática del Contenido

El instalador descargará los 13 GB de mapas desde Google Drive. Esto puede tomar tiempo dependiendo de tu conexión.

Características de la descarga:

  • Resumible: Si se interrumpe, continúa desde donde quedó
  • Verificación: Checksum de cada archivo descargado
  • Extracción automática: El .tar.gz se extrae y elimina
  • Optimización de espacio: Solo ocupa 13 GB finales
Proceso interno del instalador: 1. Descarga Atlas_Linux_v1.0.0.tar.gz (13 GB) 2. Extrae a la carpeta seleccionada 3. Verifica integridad de todos los archivos 4. Elimina el archivo .tar.gz para ahorrar espacio 5. Crea accesos directos y configuración
5

Finalización y Acceso

Cuando termine la instalación, tendrás estas opciones:

Opciones al finalizar: ✓ Ejecutar Atlas Interactivo ahora ✓ Crear entrada en el menú de aplicaciones ✓ Crear acceso directo en escritorio ✓ Abrir carpeta de instalación

Estructura de archivos creada:

# Ejemplo de estructura:
~/AtlasInteractivo/
├── Atlas_Interactivo # Ejecutable principal
├── atlas.desktop # Entrada para el menú
├── icon.png # Icono de la aplicación
├── assets/ # 174k+ archivos de mapas
├── .version.json # Información de versión
└── README # Documentación

Para ejecutar manualmente:

$ ~/AtlasInteractivo/Atlas_Interactivo
# O usa el acceso creado en el menú de aplicaciones

Configuración Avanzada

Integración con el Sistema

Crear enlace simbólico global

# Crea un enlace en /usr/local/bin para acceso desde cualquier terminal
$ sudo ln -s ~/AtlasInteractivo/Atlas_Interactivo /usr/local/bin/atlas
$ atlas # Ahora puedes ejecutar desde cualquier lugar

Actualizar el instalador AppImage

Si necesitas una nueva versión del instalador:

$ rm AtlasInstaller.AppImage
$ wget "NUEVA_URL" -O AtlasInstaller.AppImage
$ chmod +x AtlasInstaller.AppImage

Solución de Problemas

Error: "bash: ./AtlasInstaller.AppImage: Permiso denegado"

$ chmod +x AtlasInstaller.AppImage
# Si no funciona, prueba con:
$ sudo chmod +x AtlasInstaller.AppImage

Error: "FUSE: Error al montar AppImage"

Instala libfuse:

# Ubuntu/Debian:
$ sudo apt install libfuse2

# Fedora:
$ sudo dnf install fuse

# Arch Linux:
$ sudo pacman -S fuse2

El instalador no se abre (sin errores)

Ejecuta en terminal para ver mensajes de error:

$ ./AtlasInstaller.AppImage --debug
# O para AppImages en general:
$ ./AtlasInstaller.AppImage --appimage-extract-and-run

No hay suficiente espacio en disco

# Verifica espacio disponible:
$ df -h ~

# Necesitas al menos 15 GB libres. Para liberar espacio:
$ sudo apt autoremove # Ubuntu/Debian
$ sudo dnf autoremove # Fedora
$ sudo pacman -Sc # Arch

Problemas con SELinux (Fedora/RHEL)

# Temporalmente deshabilitar SELinux para la ejecución:
$ ./AtlasInstaller.AppImage --appimage-extract
$ cd squashfs-root
$ ./AppRun

# O agregar contexto SELinux:
$ sudo chcon -t unconfined_exec_t AtlasInstaller.AppImage

Sistema de Actualizaciones

Atlas Interactivo en Linux se actualiza automáticamente:

Funcionamiento de actualizaciones: 1. La aplicación verifica versiones al iniciar 2. Descarga solo archivos modificados (MBs, no GBs) 3. Aplica parches automáticamente 4. Notifica cuando está lista
Actualización manual: Si necesitas forzar una actualización:
$ ~/AtlasInteractivo/Atlas_Interactivo --check-updates
# O elimina .version.json para forzar actualización completa
