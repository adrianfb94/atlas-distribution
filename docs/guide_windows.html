Guía de Instalación para Windows

Instala Atlas Interactivo en Windows 10/11 de 64-bit. Solo necesitas hacer clic una vez y nuestro instalador hará el resto.

Requisitos del Sistema

Antes de comenzar, verifica que tu equipo cumple con estos requisitos:

Sistema Operativo

Windows 10 o 11 (64-bit)

Memoria RAM

Mínimo 4 GB

Espacio en Disco

25 GB libres

Consejo: Instala en una unidad SSD para mejor rendimiento. Si usas HDD, la primera carga puede ser más lenta.
1

Descarga el Instalador

Descarga el archivo AtlasInstaller.exe (5 MB) desde nuestra página principal:

Tamaño: 5 MB
Nombre: AtlasInstaller.exe
Tipo: Aplicación Windows
Seguridad: Windows Defender puede mostrar una advertencia porque el archivo no está firmado. Esto es normal para software de código abierto. Haz clic en "Más información" y luego "Ejecutar de todos modos".

Ubicación recomendada: Descarga el instalador en tu Escritorio o en la carpeta de Descargas para encontrarlo fácilmente.

2

Ejecuta el Instalador

Haz doble clic en AtlasInstaller.exe para comenzar.

Interfaz del instalador: Verás una ventana con:

  • Selector de carpeta de instalación
  • Barra de progreso de descarga
  • Información de espacio disponible
  • Botón "Instalar"
Consejo: Si tienes múltiples discos, elige uno con al menos 25 GB libres. Evita instalarlo en C:\Program Files si no tienes permisos de administrador.
3

Selecciona Carpeta de Instalación

Elige dónde quieres instalar Atlas Interactivo. Recomendamos:

C:\Users\[TuUsuario]\AtlasInteractivo\
D:\AtlasInteractivo\
(Cualquier carpeta con 25 GB libres)

El instalador hará lo siguiente:

  1. Creará la carpeta seleccionada
  2. Descargará los 20 GB desde Google Drive (resumible)
  3. Extraerá automáticamente el archivo .zip
  4. Eliminará el archivo .zip para ahorrar espacio
Importante: No cierres el instalador durante la descarga. Si se interrumpe, puedes reanudar desde donde quedó.
4

Espera la Descarga y Extracción

El proceso puede tomar entre 30 minutos y varias horas dependiendo de tu conexión a Internet.

Durante la instalación verás:

  • Barra de progreso de descarga (20 GB total)
  • Velocidad de descarga en tiempo real
  • Tiempo restante estimado
  • Verificación de integridad de archivos
Consejo: Puedes minimizar el instalador y seguir usando tu computadora normalmente. El proceso continúa en segundo plano.
5

Finalización y Primer Ejecución

Cuando el instalador termine, verás un mensaje de éxito.

Opciones disponibles:

  • Ejecutar Atlas Interactivo ahora: Inicia la aplicación inmediatamente
  • Abrir carpeta de instalación: Muestra dónde se instaló
  • Crear acceso directo en escritorio: Agrega un icono para fácil acceso

Archivos creados:

[Carpeta de instalación]/
├── Atlas_Interactivo.exe # Ejecutable principal
├── assets/ # 174k+ archivos de mapas
├── .version.json # Información de versión
└── README.txt # Información básica

Solución de Problemas

Error: "No hay suficiente espacio en disco"

Verifica que tengas al menos 25 GB libres. El instalador necesita espacio para:

  • Archivo .zip temporal (20 GB)
  • Archivos extraídos (20 GB)
  • Total temporal: 40 GB durante la instalación
  • Final: Solo 20 GB ocupados

Windows Defender bloquea la descarga

Solución:

  1. Abre "Seguridad de Windows"
  2. Ve a "Protección contra virus y amenazas"
  3. Haz clic en "Historial de protección"
  4. Busca AtlasInstaller.exe y haz clic en "Permitir en dispositivo"

La descarga se interrumpe

Simplemente ejecuta el instalador nuevamente. Google Drive permite descargas resumibles, por lo que continuará desde donde se detuvo.

Sistema de Actualizaciones

Atlas Interactivo se actualiza automáticamente. No necesitas volver a descargar 20 GB.

Cuando haya una actualización:

  1. La aplicación verificará automáticamente al iniciar
  2. Descargará solo los archivos modificados (MBs, no GBs)
  3. Aplicará los cambios en segundo plano
  4. Te notificará cuando esté lista
Consejo: Mantén la aplicación abierta para recibir actualizaciones automáticas. Si la cierras, se actualizará la próxima vez que la abras.
