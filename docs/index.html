Preguntas Frecuentes

¿Necesito 40GB para instalar? (20GB zip + 20GB extraído)

NO. Nuestro instalador extrae y elimina el archivo .zip/.tar automáticamente. Solo necesitas el espacio final (20GB para Windows, 13GB para Linux).

¿Cómo funcionan las actualizaciones?

El sistema mantiene un registro de versiones. Cuando hay actualizaciones, solo descarga los archivos modificados (generalmente MBs) y los aplica automáticamente.

¿Puedo pausar y reanudar la descarga?

Sí, Google Drive permite descargas resumibles. Si se interrumpe, puedes continuar desde donde quedaste.

¿Es seguro descargar desde Drive?

Sí, todos los archivos tienen checksum de verificación. El instalador valida la integridad de cada archivo antes de extraerlo.

¿Necesito permisos de administrador?

Windows: Solo si instalas en Program Files. Recomendamos carpeta de usuario.
Linux: No se requieren permisos root para AppImage.

¿Puedo mover la carpeta después de instalar?

Sí, Atlas Interactivo es portable. Puedes mover la carpeta a cualquier ubicación o incluso a otro equipo.
